# Array Utility Console

Array Utility Console is a console-based C++ application designed for working with dynamic arrays.
The program allows users to analyze, modify, compare, and manage arrays through a menu-driven interface.

This project was created as a learning exercise to practice core and intermediate C++ concepts.

---

## Features

- Manual and automatic array creation
- Printing array contents
- Statistical analysis (minimum, maximum, average, sum)
- Sorting arrays (ascending and descending)
- Replacing elements by index
- Replacing all matching elements
- Counting the number of occurrences of an element
- Reversing the array
- Swapping elements by their positions
- Replacing negative values with zero
- Replacing positive values with zero
- Saving the original array state
- Creating and managing array clones
- Comparing the current array with a saved clone
- Restoring arrays from saved states

---

## How to use

1. Run the program.
2. Create an array by choosing manual or automatic input.
3. Select actions from the menu by entering the corresponding number.
4. Follow the instructions displayed in the console.
